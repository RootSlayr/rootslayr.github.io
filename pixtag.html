PixTag

AWS-powered Serverless Image Storage and Tagging System

PixTag is an advanced serverless image storage system with tagging capabilities, developed using AWS services. The project includes image upload, tagging, and retrieval, with features such as thumbnail generation and object detection. [cite: 9, 10, 11, 12]

Features

  • Image upload and storage using AWS S3
  • Tagging functionality to organize and categorize images
  • Image retrieval based on tags
  • Thumbnail generation for efficient browsing
  • Object detection using YOLO to automatically tag objects in images

Technical Details

  • Role: Project Lead (4-person team)
  • Technologies: Python, AWS (Lambda, DynamoDB, S3), JavaScript, HTML
  • Achievements: Created APIs for efficient tag management and bulk operations, significantly enhancing image handling performance.
