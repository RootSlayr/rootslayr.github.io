MealMate

Personalized Diet Planner Android Application

MealMate is an Android application designed to improve meal planning with tailored dietary suggestions. The app uses Firebase for data storage and Google Sign-In for secure login, with an intuitive UI built using Jetpack Compose.

Features

  • Personalized dietary suggestions based on user preferences and health goals
  • Recipe recommendations with detailed nutritional information
  • Meal planning and tracking tools
  • Integration with Edamam API for extensive recipe database
  • User-friendly interface built with Jetpack Compose

Technical Details

  • Role: Lead Android Developer
  • Technologies: Kotlin, Jetpack Compose, Firebase, Edamam API
  • Achievements: Implemented a dietary suggestion algorithm with over 95% accuracy, enhancing user engagement by 20%.
